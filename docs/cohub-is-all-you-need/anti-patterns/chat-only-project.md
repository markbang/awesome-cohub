# Chat-only project

## Why it hurts
No durable truth. Restarts, new Chats, and collaborators cannot reconstruct the work.

## Do this instead
Put state in **files**, mark milestones with **Saves**, share via **Works**.

## Smell test
If closing the Chat would lose the project, it is not a Space project yet.
