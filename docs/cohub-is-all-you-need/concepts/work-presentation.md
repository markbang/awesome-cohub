# Work presentation (`hideCohubBar`)

Public Work pages can show a light Cohub host chrome (footer bar) or hide it.

| | |
|--|--|
| Meta | `hideCohubBar` |
| Entitlement | Pro / Max |
| CLI | `--hide-cohub-bar` / `--show-cohub-bar` |
| Side effect | Share/OG branding becomes more Work-first when hidden |

Not the same as your SPA’s own header/nav.
