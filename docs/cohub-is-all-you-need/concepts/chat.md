# Chat · Session

A **Chat** is a conversation context inside a Space. CLI/API name: **session**.

## Practice
- Multiple Chats for divergent goals
- Fork when exploring an alternate path
- `/skill:name` expands skills; `@space` pulls cross-space context deliberately
