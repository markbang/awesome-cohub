# Channel

External messaging surface connected to Cohub (Discord, Feishu/Lark, WeChat, etc.).

## Why it matters

Channels move **ops** out of the web UI without abandoning Space as the system of record. Files, Saves, and skills still live in the Space.

## Practice

- Prefer channel → Space prompt over “bot owns the only copy of truth”  
- Keep secrets in Space/config, not in chat logs  
- See [channel-ops](../playbooks/channel-ops.md)
