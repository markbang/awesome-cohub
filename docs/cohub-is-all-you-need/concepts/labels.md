# Labels

Lightweight tags on Spaces (and related objects) for organization and filters.

## Why it matters

Labels beat endless renaming. Pair with clear Save notes (`v0-…`) instead of churning Space titles.

## Practice

- Use labels for portfolio / status / client  
- Keep names stable; let labels change
