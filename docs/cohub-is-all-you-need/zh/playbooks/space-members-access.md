# Space 成员、角色与访问策略

## 何时

多人（或匿名访客）要进 Space，又不共享你的账号。

## 角色

| 角色 | 典型权限 |
|------|----------|
| **host** | 完整管理：成员、设置、高危操作 |
| **builder** | 在 Space 内创建/编辑项目工作 |
| **guest** | 按访问策略有限参与/查看 |

## 步骤

1. **Space 设置 → Members** 邀请并给最小角色  
2. **Access**：登录用户 / 匿名默认  
3. 用标签 + 分 Space，别把 Home 当共享垃圾桶  
4. 公开 **Work** 还有第二层权限：member ≠ workScopes  

## 完成标志

- [ ] 每人有明确角色  
- [ ] 匿名/登录默认符合分享方式
