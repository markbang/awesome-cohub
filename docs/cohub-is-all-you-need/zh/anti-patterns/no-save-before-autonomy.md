# 高自主前不存档

## 伤害
Broad cleanups and refactors become unrecoverable archaeology.

## 正确做法
Save (or fork from Save) **before** “rewrite everything” prompts. Read diffs after.

## Smell test
If you fear the next prompt, you needed a Save five minutes ago.
