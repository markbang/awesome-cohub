# Work 呈现（`hideCohubBar`）

公开 Work 可显示轻量 Cohub 宿主底栏，或隐藏。

| | |
|--|--|
| Meta | `hideCohubBar` |
| 权益 | Pro / Max |
| CLI | `--hide-cohub-bar` / `--show-cohub-bar` |
| 副作用 | 隐藏时分享/OG 更偏 Work 品牌 |

≠ 你 SPA 自己的顶栏导航。

见 [hide-cohub-bar](../playbooks/hide-cohub-bar.md)。
